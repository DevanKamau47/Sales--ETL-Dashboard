# Sales--ETL-Dashboard
# Sales Analytics Dashboard with ETL Pipeline

## Overview
This project demonstrates an end-to-end data analytics workflow:
data extraction, transformation, loading (ETL), and visualization
using a business intelligence dashboard.

## Tech Stack
- Python (pandas)
- PostgreSQL
- SQL
- Power BI
- Git/GitHub

## Architecture
CSV → Python ETL → PostgreSQL → Power BI Dashboard

## ETL Pipeline
- Extracts raw sales data from CSV
- Cleans and transforms data
- Loads structured data into a relational database

## Dashboard Insights
- Total revenue & sales trends
- Top products and categories
- Regional performance
- Monthly growth analysis

## How to Run
1. Clone repository
2. Install dependencies
3. Run `etl_pipeline.py`
4. Open Power BI file and refresh data
